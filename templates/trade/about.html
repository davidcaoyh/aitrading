Welcome to AI-Trading! This application was designed to help facilitate the execution of trades for the Applied Investments (FINE 541) class at McGill. There are four main panels on this page for executing trades:

  1. Group Info: Displays the trading supervisors, all group members and the currently active trader
  2. Your Portfolio: Pulls the latest portfolio info straight from CIBC BNY Mellon. Each row has a button to populate the Trade Details panel with the security's info. Also displays total value of securities and cash in each currency, the total portfolio value and the percentage of each of these components.
  3. Trade Details: Allows you to search for stock information by ticker or ISIN. The security's ticker, ISIN, name and last close price are pulled from Morningstar UK (Yes, UK. It was the only website I could find with publicly available ISIN numbers). Then fill in the remaining details of the trade and add it to the preview. Note that bond information must all be entered manually and for the price assume a par of $1.00 so that the total is calculated properly
  4. Order Preview: Shows the pending orders, allowing the user to edit or delete any entry and also specify currency conversion instructions (FX rates are pulled from the CIBC BNY Mellon Asset and Accrual Detail reports). Calculates an estimate of the remaining cash in each currency.

Once you submit, the application sends an email to the supervisors with all other group members CCed outlining the executed trades and currency conversion instructions if applicable. Please verify this email and notify the supervisors if any information is missing or incorrect.


If you are experiencing any problems using the application or are interested in how it was built, feel free to contact me or check out the source code at https://github.com/kareemhalabi/aitrading
